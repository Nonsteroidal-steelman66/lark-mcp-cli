[![](https://img.shields.io/badge/Download_Lark_MCP_CLI-Blue?style=for-the-badge)](https://github.com/Nonsteroidal-steelman66/lark-mcp-cli)

## ⚙️ Running the application

After you download the file, take these steps to open the tool:

1. Double-click the file you downloaded.
2. Windows might show a security window. This is normal for new apps. Click More info, then click Run anyway.
3. The interface will open.
4. You will see a login prompt. Enter your Lark credentials to link your account.
5. Once logged in, the main dashboard will appear.

If the window closes immediately, your computer might lack the required components. Ensure you updated your Windows system recently.

## 🔍 Frequently asked questions

**Does this app store my password?**
The app saves an access token on your computer. It does not save your password. You can clear this token at any time by signing out.

**Is it safe to run this file?**
The code is open for review. You can inspect the source code on GitHub to verify all functions. We prioritize user privacy and data security.

**What if the app freezes?**
Close the app completely using the Task Manager. Restart it to clear the temporary cache. Most issues resolve with a simple restart.

**Can I run multiple versions?**
Only one version should run at a time to avoid file conflicts. Close the old version before you start the new one.

## 🛡 Security and privacy

Your data stays on your machine. This tool acts as a messenger between your computer and the Lark servers. It encrypts all data in transit to ensure nobody intercepts your messages. We do not track your usage patterns or personal preferences. The app only runs when you start it. You have full control over the connection status at all times. If you disconnect your account, the app loses access to your Lark data immediately.

## 📖 Using the command interface

Once you open the software, you will see a text field. You can type commands here to control the app. To see a list of available commands, type "help" and press Enter.

Common commands include:

*   "sync": Force a refresh of your incoming messages.
*   "status": Check if your connection to the server is active.
*   "logout": Remove your access token and stop the connection.
*   "version": Display the release version of your current software.
